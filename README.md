## ⚙️Installation
### Requirements
- Node.js 14+
- npm
- ngrok (available soon)

### Steps
1. **Clone Repository**
```bash
git clone https://github.com/farrelega/Esteban.git
```
2. **Install Dependencies**
```bash
npm install
```
3. **Input ngrok Token** (available soon)
```bash
// Input token to server.js
authtoken: "TOKEN"
```
4. **Run Server**
```bash
npm start
```
